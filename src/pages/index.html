Information

Lit It Up, this game was created to introduce my coding skills, especially three.js which is a library that is able to render a 3D model. Also, this project was created by Pranchayut Netsawang, or you can call me Kimmuie, please do not copy by any means unless I give you permission because this project will be included in my portfolio. However, this project supports only desktop because of render and movement issues. And if you guys wonder how to play, there is a guide on top right while you are playing.

Lit It Up เกมนี้ถูกสร้างขึ้นเพื่อนำเสนอทักษะการเขียนโค้ดของผม โดยเฉพาะ three.js ซึ่งเป็น Library ที่สามารถแสดงผล 3D Model นอกจากนี้โปรเจ็กต์นี้จัดทำโดย ปรานต์ชยุต เนตรสว่าง หรือเรียกผมว่า คิม และได้โปรดกรุณาอย่าคัดลอกไม่ว่าด้วยวิธีใดๆ เว้นแต่ผมจะอนุญาต เพราะโปรเจ็กต์นี้จะรวมอยู่ในพอร์ตโฟลิโอของผมด้วย อย่างไรก็ตามโปรเจ็กต์นี้รองรับเฉพาะ Desktop เท่านั้นเนื่องจากปัญหาการแสดงผลและการเคลื่อนไหว และถ้าเพื่อนๆ สงสัยว่าจะเล่นยังไง ก็จะมีคำแนะนำอยู่ด้านบนขวาขณะที่คุณกำลังเล่น

How To Play

Objective
is to reach the campfire before the torch burn out (timer ran out) for lit the torch up and burning some marshmallows.

เป้าหมาย
คือการไปถึงแคมป์ไฟก่อนที่คบเพลิงจะมอดดับ
(หมดเวลา) เพื่อจุดคบเพลิงและปิ้งมาร์ชแมลโลว์

Keybind
Move Character : Click W A S D
Move Camera : Hold Right Click
Rotate Camera : Hold Middle Button
Zoom Camera : Scroll Middle Button
